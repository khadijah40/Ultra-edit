## About

This project is a static front-end clone of the UltraEdit software download page. It was built as a CSS and HTML layout exercise, focusing on replicating a professional product website with a navigation bar, hero section, and a structured multi-language download table.

---

## Features

- Responsive navigation bar with logo and menu links
- Hero section with headline and call-to-action
- Multi-language download section (English, Deutsch, Italiano, Español, Français, Português, Japanese, Korean, Chinese)
- SHA256 checksum display per download entry
- Clean, professional layout matching the original UltraEdit branding

## Getting Started

No dependencies or build tools required. Just open the project in a browser.

```bash
# Clone the repository
git clone https://github.com/khadijah40/Ultra-edit.git

# Navigate into the project folder
cd Ultra-edit

# Open in your browser
open index.html
```

Or simply drag and drop `index.html` into any browser window.

## Deployment

This project is deployed on **Netlify** via drag-and-drop static hosting.

To deploy your own version:

1. Fork or clone this repository
2. Go to [netlify.com](https://netlify.com) and log in
3. Click **Add new site → Deploy manually**
4. Drag the project folder into the upload area
5. Your site will be live instantly with a Netlify URL
